# Sprint 2: Wow-réteg Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** A jóváhagyott wow-réteg leszállítása: smoke-gate, ghost-tipográfia, lime kiáltvány-sáv, DecodeText, scroll-reveal, route-transition, boot-szekvencia, per-route accentek.

SSR mindig a végállapotot adja. Spec: `docs/superpowers/specs/2026-06-10-sprint2-wow-design.md`.

**Tech Stack:** Next.js 15 App Router · React 19 · Tailwind v4 · Puppeteer (devDep, smoke)

### Task 4: Scroll-reveal rendszer

**Files:** Create: `src/components/RevealObserver.tsx` (client) · Modify: `src/app/layout.tsx` (mount + inline data-js script), `src/app/globals.css`, attribútumok: `src/app/page.tsx` (STATS cellák, inline meter, PRINCIPLES article-k, project index sorok, PROCESS cellák), `src/app/method/page.tsx` (ladder sorok, DELIVERABLES article-k), `src/app/work/page.tsx` (archive article-k)

- [ ] layout.tsx `<body>` elejére inline script (no-JS guard): `<script dangerouslySetInnerHTML={{ __html: "document.documentElement.setAttribute('data-js','')" }} />`
- [ ] RevealObserver: useEffect-ben EGY `IntersectionObserver` (`threshold: 0.15`, `rootMargin: "0px 0px -8% 0px"`), minden `[data-reveal]`-t observe-ol, belépéskor `data-inview="true"` + unobserve. MutationObserver NEM kell (route-váltásnál a komponens pathname-re újrafuttatja a scant — `usePathname` dep).
- [ ] globals.css:

```css
/* Scroll-reveal — hidden state ONLY when JS is live and motion allowed */
@media (prefers-reduced-motion: no-preference) {
  html[data-js] [data-reveal]:not([data-inview="true"]) {
    opacity: 0;
    clip-path: inset(0 100% 0 0);
  }
  html[data-js] [data-reveal] {
    transition:
      clip-path 0.6s cubic-bezier(0.2, 0.8, 0.2, 1),
      opacity 0.45s ease;
  }
}
```

- [ ] Attribútum-minta: `data-reveal` + stagger inline `style={{ transitionDelay: \`${i * 70}ms\` }}`.
- [ ] Commit: `feat(motion): shared IntersectionObserver scroll-reveal with clip-path wipe`
